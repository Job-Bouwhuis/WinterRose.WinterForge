enum WinterRose.WinterForgeSerializing.Workers.OpCode

Operation codes that the deserialization process for WinterForge understands.

Enumerator
DEFINE 

Defines an object.

EG "0 System.Numerics.Vector3 0 0"
0 - define
type name
object ID of 0
0 items from the stack for constructor arguments

SET 

Sets the value of a field on the current object

EG "1 X 25"
1 - SET
X - field name
25 - value

END 

Ends the working on the most recent defined object.

EG "2 0"
2 - END
0 - object key its ending. this must match the most recent defined object that has not yet had this end operation called on it.

PUSH 

pushes a value to the stack. used for constructor and method calls

EG "3 5"
3 - PUSH
5 - value. value can also be a _ref() or _stack() call

CALL 

Calls a method. this opcode has not yet been fully designed. therefor an example can not yet be given.

ELEMENT 

Adds an element to the current list

EG "5 _ref(4)"
5 - ELEMENT
_ref(4) - value, the reference of an object with ID 4

LIST_START 

begins creating a new list

EG "6 System.Numerics.Vector3"
6 - LIST_START
list item type

LIST_END 

Ends a list and puts it to the stack. can then be accessed by SET using '_stack()'

Has no arguments. line should be only "7".

RET 

tells the InstructionExecutor to return the object of which key is given

EG "8 4"
8 - RETURN
4 - key of which object reference to return

PROGRESS 

invokes InstructionExecutor.ProgressMark

ACCESS 

Attempts to access the top stack value. on success puts the value on the stack. on fail, exception

EG: "10 Player"

SETACCESS 

Sets the value on the top stack value, rather than the current working object

refer to SET for the example

AS 

Takes the top stack item and puts it as a reference item on the given ID (can be used to override an id reference)

EG: 12 0
12 - AS
0 - store the stack item on ID 0. even if an object at that ID already exists
